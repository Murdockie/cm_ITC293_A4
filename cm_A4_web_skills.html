Chris Murdock

I’m a passionate web developer specialising in crafting responsive, accessible, and performance driven websites. From semantic HTML and modern CSS layouts to dynamic JavaScript interactions and SEO best practices, I build experiences that delight users and meet business goals.

Web Development Overview

Over the past, I have worked on a variety of web projects ranging from small business landing pages to complex single-page applications. My workflow begins with a user-centered design approach— identifying user needs and mapping out intuitive information architectures. I then translate designs into clean, maintainable code, ensuring pixel-perfect layouts across devices using CSS Flexbox and Grid.

Core Competencies

HTML5 & CSS3: I leverage modern HTML elements for semantic structure and advanced CSS features like custom properties, transitions, and animations to create engaging interfaces.

JavaScript & Frameworks: With vanilla JavaScript at the core, I’ve built dynamic features such as form validations, interactive components, and modular codebases. I’m also experienced with React.js for building scalable, component driven applications.

Responsive Design: Using mobile first principles, I ensure layouts adapt gracefully to any screen size employing media queries, fluid grids, and flexible images.

Design & Accessibility

I prioritise accessibility by using ARIA roles, proper heading hierarchy, and keyboard navigation to ensure all users can interact with content. I also employ colour contrast checks and semantic markup to support screen readers.

Collaboration with the designers is key: I translate Figma prototypes into code, maintain designs consistency, and provide feedback to improve usability and visual hierarchy.

Performance & SEO

Optimising page load and runtime performance is a core focus. I implement techniques like code splitting, lazy loading images, and minification to deliver fast experiences. I also structure content for SEO using meaningful titles, meta tags, and schema markup to improve search Visibility.

Tooling & Workflow

My development environment includes version control with Git, task automation via npm scripts, and bundlers like Webpacks. I write clean, modular code and utilise formatters to maintain consistency.

I’m comfortable with headless CMS integrations and deploying sites using platforms like Netlify for continuous integration and delivery.
